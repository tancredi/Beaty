# Beaty

> A simple and playful HTML5 loop machine using WebRTC for audio recording

## Warning

* ** This repo is unfinished and currently under development. ** *

## Setup

    git clone git@github.com:tancredi/Beaty.git
    cd Beaty
    npm install

## Build

    npm run build

## Serve

    npm run serve

## Watch

    npm run watch
